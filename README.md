# 关于 gin-web-F

gin-web-F 是基于[Gin](https://github.com/gin-gonic/gin) 封装的一个web框架，提供快速的业务开发

## 内置功能

- [x] core接口跨域
- [x] 支持viper配置文件解析
- [x] 支持gorm数据库组件
- [x] 数据库迁移
- [x] 支持go-redis组件
- [x] 支持RabbitMQ组件
- [x] 支持Restful API 返回规范
- [x] JWT 鉴权中间件
- [x] 数据合法性validator校验
- [x] 异常数据统一捕获
- [x] Prometheus指标记录
- [x] 接口rate限流
- [x] 支持Graphql
- [ ] 日志收集
- [x] cron定时任务
- [ ] 支持websocket
